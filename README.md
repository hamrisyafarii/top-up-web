# 🚀 GameTopUp — Digital Game Top-Up Platform

A fullstack SaaS web application that enables users to top up in-game currency quickly, securely, and efficiently.

## 🧠 Problem
Many gamers struggle with:
- Complicated top-up processes
- Limited payment options
- Lack of secure and fast transactions

## 💡 Solution
GameTopUp simplifies the process by providing:
- Instant top-up services
- Secure payment integration with Xendit
- Authentication system for user management

## ⚙️ Tech Stack
### Backend
- NestJS
- Prisma ORM
- PostgreSQL
- Xendit (Payment Gateway)
- Passport (Authentication)

### Frontend
- React.js
- Axios
- TanStack React Query
- Zod
- ShadCN UI

## ✨ Features
- 🔐 User Authentication
- 💳 Secure Payment Integration (Xendit)
- 📦 Order Management System
- ⚡ Real-time transaction handling
- 📊 Dashboard for tracking transactions

## 🏗️ Architecture
- Fullstack separated (Frontend & Backend)
- REST API based communication
- Webhook handling for payment updates
