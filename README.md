# 🚀 GameTopUp SaaS Platform

## 📌 Overview
A fullstack SaaS application that simplifies game top-up transactions by providing secure authentication, integrated payment processing, and real-time transaction management.

## ⚙️ Tech Stack
- Frontend: React.js
- Backend: Node.js / NestJS
- Database: PostgreSQL + Prisma
- Auth: Passport
- Others: Xendit, React Query, Zod

## ✨ Features
- User Authentication
- CRUD Functionality
- Payment Integration (if any)
- Search / Filter
